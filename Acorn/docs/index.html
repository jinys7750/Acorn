Customize Style

Note: Mimity is built with development tools: NPM, Gulp, Mustache

In order to be able to setup Mimity development environment you will need to install Node.js and its Package Manager (npm). Read here.

When Node.js and npm is installed you are ready to install all dependencies necessary for Mimity development.

npm install

This will start the process of installing all dependencies which is listed inside package.json file. Wait until installation is finished.

After installaltion is finished, feel free to edit the file related to the component you want to customize. All source files are stored in the folder 'src'

gulp watch

This will build all files changed from 'src' folder and generate it to 'build' folder

* If there is a new version dependencies and you update it, do not forget to run the gulp copy-module command, this will copying required dependencies from 'node_modules' to 'build' folder
